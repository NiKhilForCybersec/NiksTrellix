Proper sizing is critical for ePO performance. Undersized infrastructure leads to slow console response, delayed policy enforcement, and event processing backlogs.

Storage Considerations

Database storage grows with event retention. Calculate approximately 1-2GB per 1,000 endpoints with 90-day retention. Master repository requires 50-100GB for full product catalog.

Agent Handler Sizing

Each Agent Handler can support approximately 10,000 agents under normal conditions. Consider:

  • WAN Bandwidth â€” Handlers reduce traffic to central ePO
  • Event Volume â€” High-threat environments may require additional handlers
