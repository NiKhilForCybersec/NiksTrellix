VSE End of Life

VirusScan Enterprise 8.8 reached end of life in December 2021. Migrate to ENS as soon as possible for continued protection and support.

Assess Current State
  • Inventory VSE versions deployed
  • Document custom exclusions and policies
  • Identify any VSE Access Protection rules
Prepare ePO
  • Upgrade ePO to 5.10+ if needed
  • Check in ENS packages
  • Create ENS policies (can import some VSE settings)
Pilot Deployment
  • Deploy to 50-100 test systems
  • Monitor for compatibility issues
  • Validate exclusions work correctly
Mass Migration
  • Use ePO deployment task (handles VSE removal)
  • ENS installer auto-uninstalls VSE
  • Deploy in waves to manage risk
VSE to ENS Migration Command 
# ENS installer handles VSE removal automatically
# No separate uninstall needed

# Via ePO: Product Deployment task
# Select "Endpoint Security Platform" + modules
# ENS detects VSE and migrates settings

# Manual (if needed):
setupEP.exe /qn ADDLOCAL=tp,fw,wc REMOVE_VSE=1

Symantec DLP to Trellix DLP

Migration from Symantec DLP to Trellix DLP requires careful planning as policy structures differ between platforms.

Component Mapping

Symantec DLP Trellix DLP Migration Notes
Enforce Server ePO + DLP Extension Central management moves to ePO
Detection Server DLP Discover Similar scanning capabilities
Network Prevent DLP Prevent MTA/ICAP integration similar
Endpoint Prevent DLP Endpoint Requires McAfee Agent
IDM (Fingerprinting) Content Fingerprinting Re-register documents
EDM (Database) Database Fingerprinting Re-fingerprint databases

Policy Recreation

Symantec DLP policies cannot be directly imported. Document all policies, classifications, and response rules, then recreate in Trellix. Use the migration as an opportunity to simplify and optimize.

Third-Party AV Migration

Identify Existing AV

Common products: Symantec Endpoint Protection, Trend Micro, Sophos, CrowdStrike, Windows Defender

Deploy McAfee Agent

Install McAfee Agent first. Can coexist with other AV temporarily.

Uninstall Existing AV

Use vendor removal tools for clean uninstall. Reboot if required.

Deploy ENS

Deploy Trellix ENS via ePO once previous AV is removed.

Common AV Removal Tools 
# Symantec
CleanWipe.exe (download from Broadcom)

# Trend Micro
Trend Micro Diagnostic Toolkit

# Sophos
Sophos Zap removal tool

# Windows Defender (disable, don't remove)
Set-MpPreference -DisableRealtimeMonitoring $true

On-Premises to ePO SaaS Migration

Provision SaaS Tenant
Work with Trellix to provision your ePO SaaS tenant.
Export Policies
Export policies from on-prem ePO. Some manual recreation may be needed.
Re-Deploy Agents
Deploy new agent package pointing to cloud handlers. Cannot migrate existing agents.
Decommission On-Prem
Once all agents report to SaaS, decommission on-prem infrastructure.
