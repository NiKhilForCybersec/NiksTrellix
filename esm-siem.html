Data Sources

Firewalls, routers, switches, Linux/Unix systems. Configure to send syslog to Receiver IP on UDP/TCP 514.

Windows Event Collector (WEC) or agent-based collection. Security, System, Application logs.

Cloud services via API: AWS CloudTrail, Azure Activity, O365, custom applications.

Correlation Rules

Example: Brute Force Detection
# Rule: Multiple Failed Logins from Same Source
Condition: 
  Event Type = Authentication Failure
  Count >= 5
  Timeframe = 5 minutes
  Group By = Source IP

Action:
  Generate Alert (High Severity)
  Add to Watchlist
  Trigger Case Creation
