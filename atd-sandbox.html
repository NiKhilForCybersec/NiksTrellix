ATD Overview

Trellix ATD is an on-premises sandboxing solution that analyzes suspicious files in isolated virtual environments. It detects malware, zero-day exploits, and advanced threats through behavioral analysis.

# Submit file for analysis
curl -X POST https://atd-server/php/fileupload.php \
  -H "VE-SDK-API: {session_id}" \
  -F "file=@suspicious.exe" \
  -F "data={\"submitType\":\"0\",\"analysisProfile\":\"1\"}"

# Check analysis status
curl -X GET "https://atd-server/php/samplestatus.php?id=12345" \
  -H "VE-SDK-API: {session_id}"

# Get report
