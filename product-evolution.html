Product Naming Evolution

Understanding product name changes from McAfee to Trellix.

Product Name Mapping

Following the STG acquisition and merger of McAfee Enterprise and FireEye, products were rebranded under the Trellix name. Use this reference to understand legacy naming.

Current Name (Trellix) Previous Names Notes
Trellix ePO McAfee ePO, McAfee ePolicy Orchestrator Core management platform
Trellix Endpoint Security (ENS) McAfee ENS, McAfee Endpoint Security Replaced VirusScan Enterprise
Trellix EDR MVISION EDR, McAfee EDR, FireEye HX Combined products
Trellix XDR FireEye Helix, MVISION XDR XDR platform
Trellix DLP McAfee DLP, McAfee Total Protection for DLP Data protection suite
Trellix Agent McAfee Agent, MA Endpoint communication
Trellix ATD McAfee ATD, Advanced Threat Defense Sandbox analysis
Trellix TIE McAfee TIE, Threat Intelligence Exchange Reputation sharing
Trellix ESM McAfee ESM, Enterprise Security Manager SIEM platform
Trellix NSP McAfee NSP, Network Security Platform IPS appliances
