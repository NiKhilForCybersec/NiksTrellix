What is TIE?

Threat Intelligence Exchange (TIE) enables real-time sharing of file and certificate reputation data across your security ecosystem. When one endpoint encounters a threat, all endpoints learn about it instantly.

DXL (Data Exchange Layer)

DXL is the messaging fabric that enables real-time communication between security products. Think of it as a pub/sub system for security events.

FeatureDescription
Pub/Sub MessagingProducts publish and subscribe to security events
Service InvocationRequest/response for reputation queries
Scalable FabricMultiple brokers for high availability
Open SDKPython, Java, JavaScript clients available

TIE Reputation Flow

File Encountered
Endpoint encounters unknown file. ENS calculates hash.
TIE Query
ENS queries TIE server via DXL for file reputation.
Reputation Check
TIE checks local DB, GTI, and any custom reputations.
Action Taken
Based on reputation: Allow, Block, or send to ATD for analysis.
