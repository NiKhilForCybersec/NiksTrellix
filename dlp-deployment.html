DLP Deployment Guide

Step-by-step deployment of Trellix DLP Endpoint, Network, and Discover.

Deployment Order

Install DLP Extension
Check in DLP extension to ePO. Configure evidence server location.
Create Classifications
Define data classifications before deploying agents. Test with sample data.
Create Policies
Build DLP policies in Monitor mode initially. Assign to test groups.
Deploy DLP Endpoint
Deploy DLP agent via ePO. Requires McAfee Agent already installed.
Monitor & Tune
Review incidents, adjust policies, add exceptions. Then enable blocking.

Prerequisites

DLP Endpoint requires McAfee Agent 5.7+ and .NET Framework 4.7.2+. Deploy these first if not present.

Network DLP

DLP Prevent Deployment

  1. Deploy appliance (physical or virtual)
  2. Configure network interface
  3. Register with ePO server
  4. Configure email gateway (MTA) integration
  5. Configure web proxy (ICAP) integration
  6. Apply DLP policies

DLP Monitor Deployment

  1. Deploy appliance on network TAP or SPAN port
  2. Configure capture interface
  3. Register with ePO server
  4. Apply DLP policies (detection only)
