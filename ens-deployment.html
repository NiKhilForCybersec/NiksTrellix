Deployment Prerequisites

System Requirements

  • OS: Windows 10/11, Windows Server 2016+, macOS 10.15+, Linux (RHEL, Ubuntu)
  • RAM: 2 GB minimum (4 GB recommended)
  • Disk: 2 GB free space
  • Agent: McAfee Agent 5.7.x or later

Deployment Methods

Deploy via ePO

Check In Packages
Menu ‚Üí Software ‚Üí Check In Package. Check in all ENS modules.
Create Deployment Task
Client Tasks ‚Üí New Task ‚Üí Product Deployment. Select ENS modules.
Assign to Systems
Assign task to System Tree groups. Set schedule (immediate or maintenance window).
Monitor Progress
Dashboards ‚Üí Deployment. Monitor success/failure rates.

Standalone Installation

Command Line Install 
# Install all modules silently
setupEP.exe /qn ADDLOCAL=ALL

# Install specific modules
setupEP.exe /qn ADDLOCAL=tp,fw,wc,atp

# With logging
setupEP.exe /qn /l*v C:\temp\ens_install.log

SCCM / Intune Deployment

  1. Export ENS package from ePO (includes agent)
  2. Create SCCM Application or Package
  3. Set install command: setupEP.exe /qn ADDLOCAL=ALL
  4. Set detection rule: Check for mfetp.exe in Program Files
  5. Deploy to collection

Phased Rollout Strategy

Phase Scope Duration Purpose
Pilot 50-100 systems (IT staff) 1-2 weeks Identify issues, tune policies
Phase 1 500-1000 systems (early adopters) 2-3 weeks Broader testing, refine exclusions
Phase 2 25% of environment 2-3 weeks Scale testing
Phase 3 Remaining systems 2-4 weeks Full deployment

Post-Deployment Validation

Validation Commands 
# Check ENS version
reg query "HKLM\SOFTWARE\McAfee\Endpoint\Security\Platform" /v Version

# Check services running
sc query mfetp
sc query mfefwc
sc query mfewc

# Check DAT version
reg query "HKLM\SOFTWARE\McAfee\AVEngine" /v AVDatVersion

# Force policy update
cmdagent.exe -e

# Test detection (EICAR)
echo X5O!P%@AP[4\PZX54(P^^)7CC)7}$EICAR-STANDARD-ANTIVIRUS-TEST-FILE!$H+H* > test.txt
