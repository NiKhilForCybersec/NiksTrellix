What is Trellix Endpoint Security?

Trellix Endpoint Security (ENS) is a comprehensive endpoint protection platform that provides multi-layered defense against malware, exploits, and advanced threats. ENS evolved from McAfee VirusScan Enterprise (VSE) and represents a complete architectural redesign with modern threat detection capabilities.

â„¹ï¸
ENS vs VSE

ENS replaced VirusScan Enterprise (VSE) which reached end-of-life. ENS offers improved performance, better threat detection, and a modular architecture that allows selective deployment of protection features.

Product Evolution

2024 - Current

Trellix ENS 10.7.x

Trellix branding, enhanced cloud integration, improved ATP with machine learning, XDR telemetry support.

2020 - 2023

McAfee ENS 10.7

Major release with Exploit Prevention, Real Protect cloud scanning, and improved AMSI integration.

2017 - 2020

McAfee ENS 10.5/10.6

Initial ENS releases replacing VSE, modular architecture introduced, Adaptive Threat Protection added.

Legacy

VirusScan Enterprise 8.8

Final VSE version, end-of-life December 2021. Organizations should migrate to ENS.

ENS Modules

ENS uses a modular architecture allowing organizations to deploy only the protection features they need. All modules are managed through ePO and share a common platform.
