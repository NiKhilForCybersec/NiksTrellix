Backup & Recovery

Backup Components

  • SQL Database ‚Äî Full backup daily, transaction log every 15-30 minutes
  • Keystore ‚Äî C:\Program Files (x86)\McAfee\ePolicy Orchestrator\Server\keystore
  • Server.ini ‚Äî Server configuration file
  • SSL Certificates ‚Äî Console and agent communication certs
  • Extensions ‚Äî Custom extensions (typically re-download from master)
SQL Backup Script 
-- Full backup
BACKUP DATABASE [ePO_SERVERNAME] 
TO DISK = N'E:\Backups\ePO_Full.bak' 
WITH COMPRESSION, INIT;

-- Transaction log backup
BACKUP LOG [ePO_SERVERNAME] 
TO DISK = N'E:\Backups\ePO_Log.trn' 
WITH COMPRESSION;

Database Maintenance

Index Maintenance 
-- Reorganize fragmented indexes
ALTER INDEX ALL ON [dbo].[EPOEvents] REORGANIZE;

-- Update statistics
UPDATE STATISTICS [dbo].[EPOEvents];

-- Check database integrity
DBCC CHECKDB ('ePO_SERVERNAME') WITH NO_INFOMSGS;
