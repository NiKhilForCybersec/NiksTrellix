What is Hybrid Deployment?

Hybrid deployment combines ePO SaaS cloud management with on-premises agent handlers. This provides the benefits of cloud management while maintaining local infrastructure for bandwidth optimization and resilience.

Legacy Support

Deployment Steps

Configure ePO SaaS
Set up your ePO SaaS tenant and configure base policies.
Deploy On-Prem Handlers
Install agent handlers at each site. Register them with ePO SaaS.
Configure Repositories
Set up distributed repositories at each site to cache updates locally.
Deploy Agents
Install agents pointing to local handlers. Configure fallback to cloud handlers.
