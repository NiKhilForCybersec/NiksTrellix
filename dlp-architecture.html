What is Trellix DLP?

Trellix Data Loss Prevention (DLP) protects sensitive data from unauthorized disclosure across endpoints, network, and storage systems. DLP provides content inspection, data classification, and policy enforcement to prevent accidental or malicious data leakage.

DLP Endpoint

DLP Endpoint protects sensitive data on workstations and laptops by monitoring and controlling data movement through various channels.

Protected Channels

Channel Description Actions
Removable Storage USB drives, external HDD, SD cards Block, Encrypt, Monitor, Justify
Cloud Services Dropbox, OneDrive, Google Drive, Box Block, Monitor, Justify
Email Clients Outlook, Thunderbird, webmail Block, Encrypt, Monitor, Justify
Web Upload HTTP/HTTPS file uploads Block, Monitor
Printing Local and network printing Block, Monitor, Watermark
Clipboard Copy/paste between applications Block, Monitor
Screen Capture Screenshots, screen recording Block, Monitor
Network Share File copy to network locations Block, Monitor, Justify

DLP Monitor

Passive deployment for detection and forensic capture without blocking.

Deployment:
  • Network TAP or SPAN port connection
  • Passive inspection of all traffic
  • Detection-only mode (no blocking)
  • Full packet capture for forensics
Use Cases:
  • Policy tuning before enabling blocking
  • Compliance monitoring and reporting
  • Forensic investigation support
  • Shadow IT detection

DLP Discover

DLP Discover scans data repositories to find and classify sensitive data at rest.

Supported Repositories

On-Premises:
  • Windows File Servers (CIFS/SMB)
  • SharePoint On-Premises
  • SQL Server Databases
  • Oracle Databases
  • Exchange Mailboxes
Cloud:
  • Microsoft 365 (OneDrive, SharePoint)
  • Box
  • Google Drive
  • AWS S3
  • Azure Blob Storage

Define Scan Targets
Configure file servers, databases, or cloud repositories to scan. Set credentials and scope.
Apply Classification
